# Política de Privacidad de YomiKomi Tana

## Resumen

YomiKomi Tana está diseñada para procesar archivos localmente. La versión actual documentada no sube mangas, cómics, webtoons, imágenes ni archivos de biblioteca a servidores externos.

## Datos recopilados por la app

La versión actual documentada de YomiKomi Tana no incluye telemetría ni recopilación automática de datos personales.

La aplicación puede leer y escribir archivos locales seleccionados por el usuario para realizar conversiones, crear salidas EPUB/CBZ, generar vistas previas o preparar biblioteca local.

## Archivos procesados

Los archivos se procesan localmente en el dispositivo del usuario. YomiKomi Tana no vende información, no distribuye archivos del usuario y no envía contenido a servidores de la app.

## Sitio y GitHub

El sitio público y las descargas se alojan en GitHub. El uso de GitHub, GitHub Pages, Issues y Releases puede estar sujeto a las políticas de privacidad y términos de GitHub.

## Funciones futuras

Si en el futuro se agregan funciones online, sincronización, telemetría, actualización automática, reportes de errores o integraciones externas, esos cambios deberán documentarse antes de publicarse.

## Contenido del usuario

El usuario es responsable de usar contenido propio, libre de derechos o autorizado. YomiKomi Tana no incluye ni distribuye manga, cómics, webtoons ni contenido protegido por derechos de autor.
