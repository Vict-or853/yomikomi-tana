# Acuerdo de Usuario Final de YomiKomi Tana

Este Acuerdo de Usuario Final regula el uso de YomiKomi Tana. Al instalar o usar la aplicación, aceptas estos términos.

## Uso permitido

Puedes usar YomiKomi Tana de forma personal y gratuita para procesar archivos locales de manga, cómics, webtoons e imágenes que sean propios, libres de derechos o que tengas autorización para usar.

## Uso prohibido

No está permitido:

- Usar la aplicación para distribuir contenido protegido sin autorización.
- Vender, sublicenciar, republicar o redistribuir YomiKomi Tana como producto propio.
- Modificar, descompilar o aplicar ingeniería inversa a binarios, instaladores o paquetes.
- Eliminar avisos de autoría, licencia, privacidad o seguridad.
- Usar la marca, logo o nombre de YomiKomi Tana para productos no oficiales.

## Contenido protegido

YomiKomi Tana no incluye manga, cómics, webtoons ni contenido protegido por derechos de autor. La aplicación solo procesa archivos locales proporcionados por el usuario.

El usuario es responsable de comprobar que tiene derecho a usar los archivos que procesa.

## Archivos del usuario

La aplicación trabaja con archivos proporcionados por el usuario. El usuario es responsable de conservar copias de seguridad de sus archivos originales y de revisar los resultados generados.

YomiKomi Tana se ofrece sin garantía absoluta de funcionamiento, compatibilidad, disponibilidad, precisión de conversión o ausencia de errores. La compatibilidad puede variar según archivos, dispositivos, sistema operativo, dependencias y configuración.
