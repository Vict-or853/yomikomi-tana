## 📖 Acerca del Proyecto

**YomiKomi Tana** es una herramienta de escritorio gratuita y 100% offline diseñada para procesar, organizar y optimizar tus archivos de manga, cómics y webtoons locales. Transforma archivos comprimidos pesados y desordenados en tomos digitales perfectos en formato **EPUB** (ideal para Kindle/Kobo) o **CBZ** (ideal para Tablets), respetando los sentidos de lectura y eliminando los molestos márgenes blancos.

---

## ✨ Características Principales

* **✂️ Auto-Crop Inteligente:** Detecta y recorta los márgenes blancos o negros inútiles para que el arte ocupe toda la pantalla de tu e-reader.
* **📏 Split Double Pages:** Divide automáticamente las páginas dobles (escanéos de dos páginas) en dos imágenes separadas para una lectura vertical perfecta.
* **📱 Especial para Webtoons:** Algoritmo de detección de cortes para transformar tiras largas de scroll infinito en páginas individuales.
* **📖 Modo Manga (RTL):** Configura la dirección de lectura de derecha a izquierda nativamente en los archivos EPUB generados.
* **📦 Carga Directa (UnRAR Nativo):** Arrastra y suelta carpetas o archivos `.ZIP`, `.RAR`, `.CBZ` y `.CBR` sin necesidad de extraerlos previamente.
* **🎨 Ajustes Visuales:** Modifica el Gamma, mantén el color original o aplica filtros de escala de grises optimizados para pantallas E-Ink.
* **🌍 Multi-idioma:** Interfaz disponible en Español, Inglés, Portugués y Japonés.

## 🚀 Descarga e Instalación

Para usuarios regulares que solo quieren usar la aplicación:

1. Ve a la pestaña de [**Releases**](../../releases) en este repositorio.
2. Descarga el archivo `YomiKomi_Setup.exe` de la versión más reciente (v1.0.0).
3. Ejecuta el instalador (creado de forma segura con Inno Setup).
4. ¡Abre YomiKomi Tana desde tu escritorio y comienza a optimizar!

> **⚠️ Nota sobre Antivirus:** Al ser un software nuevo compilado con PyInstaller, Windows Defender o tu antivirus podrían mostrar una advertencia de "Software Desconocido" (Falso Positivo). Puedes omitir la advertencia de forma segura. El procesamiento es 100% local.
