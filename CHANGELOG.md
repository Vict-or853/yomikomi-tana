# Changelog

El formato sigue [Keep a Changelog](https://keepachangelog.com/) y el proyecto usa versionado `MAJOR.MINOR.PATCH` cuando aplique.

## [Unreleased]

### Added

- Sitio oficial publicado desde `/site`.
- Documentación legal básica: `LICENSE.md`, `EULA.md`, `PRIVACY.md` y `THIRD_PARTY_NOTICES.md`.
- Plantillas de issues para bugs, funciones, app móvil y compatibilidad de dispositivos.
- Plantilla recomendada para releases.

### Changed

- Repositorio orientado a presentación pública de freeware cerrado.
- Mensajes de privacidad y seguridad más claros.
- Web pública sin capturas ni demos con contenido reconocible de terceros.

## [0.1.0-beta]

### Added

- Primera versión pública freeware.
- Conversión local.
- Página oficial.
- Documentación legal básica.

### Changed

- Repositorio orientado a freeware cerrado.
- Mensajes de seguridad más claros.

### Known limitations

- App móvil todavía no disponible.
- Instalador puede mostrar advertencia SmartScreen si no está firmado digitalmente o tiene pocas descargas.
- Compatibilidad puede variar según archivos, dependencias y dispositivos.
